# Calculadora

![](https://imgur.com/JQXircw.gif)

## Sobre

Calculadora simples de console feita em C#, contem as 4 operações básicas, soma, subtração, multiplicação e divisão.

## Funcionalidades

- Soma
- Subtração
- Divisão
- Multiplicação
- Tabuada
- Histórico de Operações
